# Health Bars Dynamic Difficulty Compat

NeoForge addon that integrates Dynamic Difficulty mob levels into Health Bars.

## Requirements

- Minecraft 1.21.1
- NeoForge 21.1.x
- Health Bars
- Dynamic Difficulty

## Build

```bash
gradlew build
```

## Usage

Drop the built jar into your `mods` folder alongside Health Bars and Dynamic Difficulty.
